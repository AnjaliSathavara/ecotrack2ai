# 🌍 EcoTrack2AI

### AI-Powered Sustainability Intelligence Platform

EcoTrack2AI is an intelligent sustainability platform that helps users measure, understand, and reduce their carbon footprint through AI-powered insights, personalized recommendations, analytics dashboards, and sustainability reporting.

---

## 🚀 Live Demo

🌐 Live Website:

https://ecotrack2ai.lovable.app

---

## 📌 Problem Statement

Many people want to live sustainably but lack awareness about how their daily habits impact the environment. Existing solutions often provide generic information without personalized guidance.

EcoTrack2AI addresses this challenge by combining carbon footprint assessment, AI-powered recommendations, sustainability analytics, and user progress tracking into a single platform.

---

## 💡 Solution

EcoTrack2AI analyzes user lifestyle habits and generates personalized sustainability insights using Artificial Intelligence.

The platform helps users:

- Measure carbon footprint
- Identify high-impact activities
- Receive AI-generated sustainability recommendations
- Track environmental progress
- Generate sustainability reports
- Build long-term eco-friendly habits

---

## ✨ Key Features

### 🌱 Carbon Footprint Assessment

Users answer questions related to:

- Transportation
- Food Habits
- Energy Consumption
- Lifestyle Choices

The platform calculates a personalized sustainability score.

---

### 🤖 AI Sustainability Assistant

Powered by Google's Gemini AI.

The assistant can:

- Answer sustainability questions
- Explain carbon emissions
- Suggest eco-friendly alternatives
- Provide personalized recommendations
- Analyze assessment results

---

### 📊 Analytics Dashboard

Interactive dashboard displaying:

- Carbon Score
- Sustainability Rating
- Impact Breakdown
- Key Environmental Metrics
- Progress Insights

---

### 📄 Sustainability Reports

Generate downloadable sustainability reports including:

- Carbon Score
- Annual Footprint Estimate
- Impact Categories
- Personalized Recommendations
- Sustainability Summary

---

### 🔐 Authentication System

Secure account management including:

- User Registration
- Login
- Logout
- Session Management
- Protected Pages
- User-Specific Data

---

### 🌗 Dark & Light Theme

Automatic modern theme support for enhanced accessibility and user experience.

---

## 🧠 AI Integration

EcoTrack2AI integrates Google's Gemini AI model to provide intelligent and personalized sustainability guidance.

The AI assistant receives user assessment context and generates recommendations based on the user's lifestyle and environmental impact profile.
